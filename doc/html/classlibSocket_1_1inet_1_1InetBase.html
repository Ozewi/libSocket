Bind the socket to a network interface, or unbind it.

Exceptions
std::system_errorif the binding fails.
Note
If a socket is bound to an interface, only packets received from that particular interface are processed by the socket.
See also
libSocket::getInterfaceList to get a list of available interfaces.
Parameters
ifaceNetwork interface name to link with, or empty to unbind the socket.
