Address * origin = nullptr )

Get a connection with a client.

If timeout is DONT_WAIT and there are no pending connections, the function returns immediately. If timeout is WAIT_DATA_FOREVER, it waits forever for an incoming connection.

◆ setListen()

void libSocket::unx::StreamServerSock::setListen ( int backlog = DEFAULT_MAX_BACKLOG)

Configure the size of the backlog and put the socket in listen mode.

The backlog argument defines the maximum length to which the queue of pending connections may grow. If a connection request arrives when the queue is full, the client may receive an error or the request may be ignored.

Exceptions
std::system_error
Parameters
backlogMaximum length of the pending connections queue [ = DEFAULT_MAX_BACKLOG ]
