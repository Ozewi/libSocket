◆ set_nodelay()

bool libSocket::SockInetStream::set_nodelay ( bool mode)

Habilitar o deshabilitar el modo de envío de datos inmediato.

Note
Cuando se activa el modo de envío inmediato, los datos son enviados al extremo sin esperar a llenar un segmento de datos. Debe activarse para aplicaciones de tiempo real.
Por omisión, el envío inmediato está deshabilitado y se usa el algoritmo de Nagle.

true: el cambio se completó con éxito | false: error
