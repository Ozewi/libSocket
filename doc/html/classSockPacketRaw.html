◆ SockPacketRaw()

SockPacketRaw::SockPacketRaw ( const char * iface,
uint16_t protocol )

Abrir el socket RAW y enlazarlo con un interface y un puerto.

Note
interface especifica el nombre del interfaz al que se enlaza el socket, en formato del sistema (ej: "eth0", "lo").
protocol es el protocolo Ethernet que incluirán los paquetes leídos y escritos (ej: ETH_P_IP == 0x0800 == IP).
Véase linux/if_ether.h
Parameters
protocol
ifaceDirección del interfaz con el que se enlaza
protocolProtocolo Ethernet
