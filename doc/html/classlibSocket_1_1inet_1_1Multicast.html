◆ join()

void libSocket::inet::Multicast::join ( const Address & group,
const std::string & iface = {} )

Join multicast group.

The network interface parameter is optional. If set, datagrams will be sent through that interface. Otherwise, the system will decide which interface to use according to the routing rules. Once the socket joins the group, it will start receiving messages addressed to the group.

Exceptions
std::system_error
Parameters
groupAddress of the multicast group to subscribe to
ifaceNetwork interface to link [= let the system decide ]

◆ setOutgoingTtl()

void libSocket::inet::Multicast::setOutgoingTtl ( int ttl)

Set TTL (Time To Live) of the outgoing multicast packets.

TTL is the maximum number of hops a packet can travel. It is decremented by 1 for each hop, and the packet is discarded when the value drops to 0.

Exceptions
std::system_error
Note
This setting only affects outgoing multicast packets.
Parameters
