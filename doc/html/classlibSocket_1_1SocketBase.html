void libSocket::SocketBase::close ( )

Close the socket.

int libSocket::SocketBase::read ( void * buffer,
int max_bytes,
int timeout = DONT_WAIT )

Read data from the socket.

If timeout == DONT_WAIT, the function reads any data pending in the input buffer and returns immediately. If timeout == WAIT_DATA_FOREVER, the function waits until all the requested data is received. Otherwise, data is read until the buffer is full or time runs out, whichever comes first.

◆ waitData()

int libSocket::SocketBase::waitData ( int timeout)

Wait for incoming data.

If timeout == DONT_WAIT, the function returns immediately, returning 0 if no data is pending and >0 otherwise. If timeout == WAIT_DATA_FOREVER, the function waits until there's data in the input queue. Otherwise, the function waits until there's any data pending or time runs out, whichever comes first.

Exceptions
std::system_errorif the handler is invalid or an error happens while waiting.
Returns
Remaining time.
0: Timeout without receiving data.

int libSocket::SocketBase::write ( const void * buffer,
int buf_len,
WriteModes write_mode = WRITE_DONT_WAIT )

Write data to the socket.

If there is no room in the system queue for the entire buffer, the function either waits for space (WRITE_WAIT_QUEUED) or writes only enough to fill the buffer (WRITE_DONT_WAIT). In both cases, the return value is the number of bytes written.
